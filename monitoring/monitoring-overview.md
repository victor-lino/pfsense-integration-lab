# Monitoring Overview

## Objective

Implement centralized infrastructure monitoring and observability for the home lab environment.

---

## Monitoring Stack

### Zabbix
Used for:
- Host monitoring
- Service monitoring
- Network visibility
- Trigger creation

### Grafana
Used for:
- Dashboard visualization
- Traffic analysis
- Real-time metrics
- Infrastructure observability

---

## Monitored Components

- pfSense firewall
- VPN traffic
- Active Directory server
- File Server
- Internal virtual machines

---

## Benefits

- Real-time visibility
- Faster troubleshooting
- Traffic monitoring
- Infrastructure health analysis
