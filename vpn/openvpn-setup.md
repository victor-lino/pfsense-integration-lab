# OpenVPN Setup

## Objective

Provide secure remote access to the internal lab infrastructure using OpenVPN.

---

## Environment

The VPN server was implemented on pfSense to allow encrypted communication between remote clients and the internal network.

---

## Main Configurations

### VPN Server
- OpenVPN configured on pfSense
- Remote access mode enabled
- Tunnel network configured

### Authentication
- User-based authentication
- Secure encrypted tunnel

### Network Access
VPN clients received access to:
- Internal DNS
- Active Directory services
- File Server
- Monitoring systems

---

## Features

- Secure encrypted communication
- Remote administrative access
- Internal resource availability
- DNS forwarding support
