# pfBlockerNG Configuration

## Objective

Implement DNS-based filtering and threat mitigation using pfBlockerNG.

---

## Features Enabled

- DNSBL filtering
- Malicious domain blocking
- Advertisement filtering
- Reputation-based protection

---

## Benefits

- Centralized web filtering
- Reduced exposure to malicious domains
- Improved browsing control inside the lab

---

## Validation

The filtering system successfully blocked:
- Advertisement domains
- Known malicious hosts
- Unwanted external connections
