# Zabbix Integration

## Objective

Monitor infrastructure services and network devices through Zabbix.

---

## Configured Hosts

The following assets were integrated into Zabbix:

- pfSense firewall
- Domain Controller
- File Server
- Internal virtual machines

---

## Monitoring Features

### Metrics
- CPU usage
- Memory utilization
- Network traffic
- Availability

### Triggers
- Host availability alerts
- Resource utilization warnings
- Connectivity monitoring

---

## Validation

Zabbix successfully collected:
- Firewall metrics
- VPN activity
- Host availability information
- Network utilization data
