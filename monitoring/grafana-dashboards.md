# Grafana Dashboards

## Objective

Visualize infrastructure and firewall metrics through dynamic dashboards.

---

## Dashboard Features

### Firewall Visibility
- WAN traffic
- LAN traffic
- VPN traffic

### Infrastructure Monitoring
- Resource utilization
- Network throughput
- Active connections

### VPN Monitoring
- Tunnel activity
- Connected clients
- Traffic analysis

---

## Benefits

Grafana dashboards provided:
- Real-time visibility
- Easier troubleshooting
- Improved observability
- Centralized monitoring experience
