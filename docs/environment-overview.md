# Environment Overview

## Project Description

This Home Lab was designed to simulate a hybrid corporate infrastructure environment focused on security, connectivity and monitoring.

The environment integrates pfSense, Active Directory, OpenVPN, DNS, File Server, Zabbix and Grafana into a centralized architecture.

---

## Main Components

### pfSense
- Edge firewall
- NAT and routing
- VPN gateway
- DNS filtering with pfBlockerNG

### Active Directory
- Domain authentication
- Group Policy Objects (GPO)
- Internal DNS resolution

### File Server
- Shared folders via UNC path
- Domain-based permissions

### OpenVPN
- Secure remote connectivity
- Access to internal resources

### Monitoring Stack
- Zabbix for infrastructure monitoring
- Grafana dashboards for visualization

---

## Environment Goals

- Simulate enterprise-grade infrastructure
- Implement secure remote access
- Practice firewall segmentation
- Centralize monitoring and observability
- Validate Active Directory integration through VPN
